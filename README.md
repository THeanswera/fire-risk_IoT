# Fire_Risk IoT-СОУЭ
Разработанная программа предназначена для автоматического расчёта индивидуального пожарного риска (ИПР) для объектов защиты в соответствии с Приказом МЧС России №1140 от 14.11.2022 «Об утверждении методики определения расчетных величин пожарного риска в зданиях, сооружениях и пожарных отсеках различных классов функциональной пожарной опасности», с возможностью сравнения традиционного варианта и варианта с учётом влияния IoT-СОУЭ (модульная система оповещения и управления эвакуацией людей на основе технологии интернета вещей).
Программа реализует расчёт ИПР по сценариям развития пожара 𝑖 и i группам людей с использованием действующих нормативных зависимостей. Для варианта IoT-СОУЭ программа дополнительно рассчитывает интегральный показатель эффективности IoT-СОУЭ (адаптивность и надёжность модулей, учёт вероятности киберинцидента) и использует его для формирования корректировки к 𝐾СОУЭ,𝑖, после чего выполняет пересчёт 𝐾п.з,𝑖	и итогового значения ИПР. Результаты отображаются в виде таблиц, метрик и сравнительной диаграммы, а также доступны для выгрузки в файлы CSV, открываемые в Excel. Программа запускается с любого стационарного или переносного электронно-вычислительного устройства, такого как телефон, ноутбук или персональный компьютер под управлением операционной системы семейства Microsoft Windows 7 и выше. 
Для реализации функционала импорта и экспорта данных в формате csv, необходима установка на компьютер программы Microsoft Excel 2003 или выше.
Принцип работы с программой, следующий: 
	После запуска исполняемого файла программы открывается ее графический интерфейс с тремя основными блоками:
	ввод исходных данных;
	настройка параметров IoT-СОУЭ;
	вывод результатов расчёта.
	Пользователь вводит/редактирует исходные данные по сценариям развития пожара i в таблице «Сценарии i»:
	Qп,i (год-1)
	tпр,i (ч/сут);
	tбл,i (мин);
	Kап,i;
	K_(обн,i),〖 K〗_(СОУЭ,i),〖 K〗_(ПДЗ,i), соответствует/ не требуется/ подтверждена».
	Пользователь задаёт состав защищаемого контингента по группам j в таблице «Группы j (по сценариям)»:
	привязка группы к сценарию i;
	наименование группы;
	временные параметры tp,i,j, tн.э,i, tск,i (мин), полученные с помощью прикладных программ Fire Dynamics Simulation (FDS).
Программа поддерживает добавление/удаление сценариев и групп управляющими кнопками, а также редактирование значений прямо в таблицах.
	Пользователь настраивает вариант с интеграцией IoT-СОУЭ в боковой панели:
	параметры адаптивности (сенсорное покрытие ключевых точек, адаптация маршрутов, автоматизация решений, время реакции);
	параметры надёжности модулей (обнаружение/локализация, основной и резервный канал связи, логика/маршрутизация, оповещение/исполнение, питание/резерв);
	вероятность киберинцидента pcyber;
	эмпирический коэффициент α, задающий влияние IoT-СОУЭ в корректировку KСОУЭ, i.
Дополнительно пользователь может выбрать конкретную группу и сценарий для оперативной настройки временных параметров через ползунки (удобно для подбора входных данных по результатам моделирования/расчётов). 
	После ввода данных программа автоматически выполняет расчёт и формирует два сопоставимых результата: 
	традиционный расчёт, включая вычисление Pпр,i и Pэ,i,j;
	расчёт с учётом IoT-СОУЭ.
На экране одновременно отображаются: R, RIoT-СОУЭ, абсолютная разница, процент снижения, а также статус выполнения условия R≤10-6 год-1.
	Пользователь анализирует результаты в нескольких представлениях:
	сравнительная диаграмма значений ИПР;
	таблица агрегирования по сценариям с итоговой строкой;
	детальная таблица расчёта по группам;
	таблица промежуточных коэффициентов по сценариям;
	разворачиваемый блок с формулами.
	Экспорт результатов выполняется нажатием кнопок выгрузки:
	CSV «построчно (группы)»;
	CSV «сценарии (коэффициенты)»;
	CSV «агрегирование».
Выгруженные файлы открываются стандартными средствами и могут быть обработаны в Microsoft Excel (или аналогах) для оформления отчётов, приложений и сравнительного анализа инженерных решений.
